A ideia desse experimento é fazer uma comparação de uma rede neural com um modelo de regressão. Peguei uma base genérica do kaggle que desrespeito ao risco 
de inadimplência de um cliente ao fazer um empréstimo, com base na sua renda. Não era a minha intenção explorar uma rede neural o máximo possível dentre as minhas 
limitações, mas sim documentar a diferença grotesca que podemos ter ao escolher um modelo errado para o nosso problema.
